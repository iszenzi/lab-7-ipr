# Find Your Pet 🐾

**Веб-сервис для поиска потерянных животных**

Find Your Pet — это современная платформа, которая помогает владельцам найти своих потерянных питомцев и объединяет людей, готовых помочь в поиске. Мы создаем сообщество неравнодушных людей, которые вместе помогают животным вернуться домой.

## 🌟 О проекте

Каждый день в России теряются сотни домашних животных. Find Your Pet решает эту проблему, предоставляя удобный и эффективный инструмент для поиска.

### Наша миссия
Мы верим, что каждый питомец заслуживает вернуться домой. Наша платформа объединяет владельцев, нашедших животных и всех, кто готов помочь в поиске.

### Ключевые преимущества

✨ **Простота использования** — интуитивный интерфейс для быстрого создания и поиска объявлений  
📍 **Точная геолокация** — интеграция с Yandex Maps для точного указания места  
🔍 **Умный поиск** — гибкая система фильтров по породе, размеру, региону и геолокации  
📱 **Доступность** — работает на всех устройствах  
🛡️ **Безопасность** — модерация контента и защита персональных данных  
📧 **Уведомления** — автоматические оповещения о новых объявлениях в вашем районе  

## 🎯 Основные возможности

### Для владельцев потерянных животных

- 📝 **Создание объявлений** — разместите информацию о вашем питомце с фотографией и описанием
- 📍 **Точное местоположение** — укажите место, где потерялся питомец, на интерактивной карте
- 🔔 **Уведомления** — получайте информацию о найденных животных в вашем районе
- 📞 **Прямая связь** — контакты для связи с теми, кто нашел вашего питомца

### Для тех, кто нашел животное

- 🐕 **Быстрое размещение** — создайте объявление о найденном животном за несколько минут
- 🔍 **Поиск владельца** — найдите объявления о потерянных животных, соответствующих описанию
- 📱 **Множество контактов** — укажите удобные способы связи (телефон, email, соцсети)

### Для всех пользователей

- 🔎 **Расширенный поиск** — фильтры по породе, размеру, цвету, региону и радиусу поиска
- 🗺️ **Карта объявлений** — визуализация всех объявлений на интерактивной карте
- 👤 **Профили пользователей** — просмотр истории объявлений и контактной информации
- 📊 **Статистика** — отслеживание успешных возвращений животных

## 🛠 Технологический стек

Find Your Pet построен на современных и надежных технологиях, обеспечивающих высокую производительность, безопасность и масштабируемость.

### Backend

- **FastAPI 0.121.0** — современный веб-фреймворк для Python с автоматической документацией API
- **PostgreSQL 17** — надежная реляционная база данных
- **SQLAlchemy 2.0.44** — мощный ORM для работы с базой данных
- **Uvicorn 0.38.0** — ASGI-сервер для запуска FastAPI приложения
- **asyncpg 0.30.0** — асинхронный драйвер для PostgreSQL
- **Python-JOSE 3.5.0** — библиотека для работы с JWT токенами
- **bcrypt 4.0.1** — безопасное хеширование паролей
- **passlib 1.7.4** — библиотека для работы с паролями
- **httpx 0.28.1** — асинхронный HTTP клиент
- **email-validator 2.3.0** — валидация email адресов
- **python-dotenv** — загрузка переменных окружения

### Frontend

- **React 19.1.1** — библиотека для создания пользовательских интерфейсов
- **React DOM 19.2.0** — рендеринг React компонентов
- **React Router DOM 7.9.4** — маршрутизация в одностраничном приложении
- **Vite 7.2.2** — быстрый инструмент сборки и разработки
- **Yandex Maps API** — интеграция карт и геолокации
- **ESLint 9.36.0** — линтер для проверки качества кода

### Инфраструктура и DevOps

- **Docker** — контейнеризация приложения
- **Docker Compose** — оркестрация многоконтейнерных приложений
- **Nginx** — веб-сервер и reverse proxy для фронтенда
- **PostgreSQL** — контейнеризированная база данных

### Безопасность

- **JWT (JSON Web Tokens)** — безопасная аутентификация без состояния
- **bcrypt** — одностороннее хеширование паролей
- **CORS** — защита от межсайтовых запросов
- **Pydantic** — валидация данных на уровне API
- **SQLAlchemy ORM** — защита от SQL-инъекций

### Архитектура

Проект следует архитектуре **клиент-сервер** с разделением на:

- **Backend API** — RESTful API на FastAPI, обрабатывающий бизнес-логику
- **Frontend SPA** — одностраничное приложение на React
- **Database** — PostgreSQL для хранения данных
- **Reverse Proxy** — Nginx для обслуживания статических файлов и проксирования API запросов

### Основные модули Backend

- **routes/** — обработчики HTTP запросов, разделенные по доменам
- **models.py** — модели данных (User, Ad)
- **schemas.py** — схемы валидации запросов и ответов
- **auth.py** — логика аутентификации и отправки email
- **database.py** — настройка подключения к БД и сессий

### Основные модули Frontend

- **pages/** — страницы приложения, каждая отвечает за отдельный экран
- **components/** — переиспользуемые UI компоненты
- **apiRequests.js** — централизованные функции для работы с API
- **ymaps.js** — интеграция с Yandex Maps для карт и геолокации

## 🔁 CI/CD (GitLab)

В репозитории настроен пайплайн GitLab CI/CD: запуск unit-тестов backend с отчётами (JUnit + cobertura), сборка frontend, сборка Docker-образов и публикация в GitLab Container Registry.

Ключевые файлы:

- .gitlab-ci.yml
- backend/tests/
- backend/requirements-dev.txt

# Настройте переменные окружения в .env файле, взяв за шаблон .env.example
# Запустите приложение
docker-compose up -d
```

После запуска приложения:
- Frontend будет доступен по адресу: http://localhost:5173
- Backend API будет доступен по адресу: http://localhost:8000
- API документация (Swagger): http://localhost:8000/docs
